# 🦖 Динозаврик (Chrome Dino Clone)

Клон классической игры из Google Chrome. Динозаврик бежит, прыгает через кактусы и уворачивается от птеродактилей.

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| `Пробел` / `↑` | Прыжок |
| `↓` | Присесть |

## 🚀 Как запустить

1. Установи Python 3 (если ещё не установлен)
2. Установи Pygame:
   ```bash
   pip install pygame
